Log ind gennem Spotify

Hvorfor skal jeg logge ind?

Spotify kræver at du giver tilladelse til at webappen kan hente dine playlister og brugerdata. Appen henter kun dine offentlige data, og gemmer dem ikke.

Når du logger ind gennem spotify kan du helt præcist se, hvilke rettigheder du giver appen. Rettighederne udløber efter time. Fortsat brug kræver så en ny login.

Hvad kan jeg se i denne webapp?

Masser af data som Spotify ikke selv viser i deres grænseflader som f.eks.

  • Popularitet - popularitet blandt Spotifys brugere
  • Tempo - angivet i BPM (Beats pr minute)
  • Energi - fra afslappet til energisk i procent
  • Dansevenlighed - angivet i procent
  • Positivitet(valence) - fra negativ 0 til positiv 100
  • Toneart incl. dur eller mol
  • Plus flere som acousticness, instrumentalness og liveness

Du kan finde data for:

  • Dine offentlige playlister
  • Dine 50 mest spillede numre i 3 tidsintervaller
  • Dine 50 mest spillede kunstnere i samme intervaller
  • Plus fri søgning i Spotifys enorme musikkatalog

For alle kunstnere kan du se popularitet på Spotify udtrykt i en procentsats. En kunstners popularitet beregnes udfra populariteten af kunstnerens numre. Derudover kan du også se de genre tags som Spotify har tildelt musikerne.

Fremvisning af data

De fleste data præsenteres i tabeller, hvor du selv kan sortere på kollonnerne. De mere kuriøse data, som f.eks. loudness, speechiness, acousticness, instrumentalness, liveness kan klikkes frem efter behov i tabellerne. Det samme kan metadata om kunstnerne.

Spotifys egne forklaringer og egne definitioner på alle begreber kan findes på engelsk nederst på siden.

Disclaimer

Spotify fastsætter intervaller, og det antal numre man kan hente. Jeg ville f.eks. rigtig gerne vise top 500 lyttede i ethvert tidsinterval. Spotify er heller ikke tydelige omkring, hvordan de genererer toplister og metadata. Så selv om de stiller et fantastisk API til rådighed, bekytter de stadig deres forretning.

Spotifys maskin- og bruger-genererede data er selvfølgelig heller ikke 100% konsistente. Tilgengæld ligger de ofte på samme niveau, men større uregelmæssigheder kan findes. F.eks. i sådan noget som toneart.
